Detailed Description

The functions themselves are arbitrary in terms of the library and are provided so that the user has a built-in API to handle data output.

Attention
This class uses the output_section class which is entirely implementation-defined by the using program. An optional definition is available for simple text output. input.
This class uses the composite_data class which is entirely implementation-defined by the using program. An optional definition is available for simple text processing.
See also
data_input provides complimentary input functionality.
