Input interface for tag-formated sections.This class provides a simple interface for parsing data that uses a tagged (e.g. XML) format. This interface must be combined virtually with an element class in order to be useful.

virtual input_receiver* tagged_input::receive_data ( data_input *  )
virtual

This function is implemented to parse a section of data. It calls the functions of managed_input to determine whether or not a portion of data contains a sub-section and creates a new one if necessary. This process is repeated until the section is completely filled as indicated by section_complete.

Reimplemented from input_receiver.

virtual bool tagged_input::is_close_tag ( data_input *  ) const
protectedpure virtual

This function is called internally to check to see if all data has been parsed for this section. The determination should be based on if the data provided is a "close" tag corresponding to the type of section the current element is.
