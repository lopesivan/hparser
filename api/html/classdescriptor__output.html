Output interface for descriptor-formated sections.This class provides a simple interface for exporting data that uses a descriptor-style format (where each section's header contains all information pertaining to the section's size.) This interface must be combined virtually with an element class in order to be useful.

virtual const output_sender* descriptor_output::send_data ( data_output *  ) const
virtual

This function is implemented to send the section's descriptor, send its own content, then pass control to its first sub-section. After all sub-sections have passed their data, control goes to the next section, and finally to its parent.
