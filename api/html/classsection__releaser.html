Detailed Description

Attention
This is a reference-counting object. It will delete the stored pointer upon its own destruction if it's the last section_releaser holding a pointer and that pointer isn't owned by something else ("owned" means part of a hierarchy but not the base.) For this reason, you should always immediately pass new elements to a section_releaser to avoid multiple groups storing pointers to the same element.
