virtual input_receiver* input_manager::import_data ( data_input *  )
virtual

This function continually calls the input_receiver function receive_input on the result of that function's return, starting with this object. Once 'this' or NULL are returned the function exits, returning that value. This allows control to be passed to new elements until completion or an error.

Implements data_importer.
